# Calendar and Journals

ZimX supports a daily workflow using date-based pages.

## Daily Notes
- Create or open a daily note from the calendar.
- Use templates to pre-fill sections.
- Keep journaling and task planning in the same system.

## Templates
- Customize your daily template.
- Use headings and checklists to guide your routine.
