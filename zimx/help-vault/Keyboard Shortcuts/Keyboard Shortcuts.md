# Keyboard Shortcuts

ZimX is designed for fast keyboard-first work.

## Common Editing
- Ctrl+B, Ctrl+I, Ctrl+K, Ctrl+U, Ctrl+T
- Ctrl+1..5 for headings
- Ctrl+7 remove heading
- Ctrl+9 clear formatting

## Navigation
- Back/Forward history
- Quick jump to page
- Search and replace

## Focus and Modes
- Read-only mode blocks edits but keeps navigation.
- Vi navigation (if enabled) keeps hands on the keyboard.
