# Search and Filtering

ZimX helps you find notes fast, whether you remember the title or just a phrase.

## Find in the Current Page
- Open the find bar and search forward or backward.
- Replace current or replace all.
- Case-sensitive search when needed.

## Search Across the Vault
- Use the global search dialog to scan all notes.
- Filters can narrow results by path or tag.

## Tags and Filters
Use tags like `@project` or `@idea` inside notes.
- Filter by tag to create a focused view.
- Combine tag filters with search for precision.
