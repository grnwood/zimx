# Advanced Setup (Tech)

This page covers power-user and multi-device setups.

## Remote Vaults
- Connect to a server-backed vault when you need access across machines.
- Read and write permissions depend on your server configuration.

## Web Client Vaults
- Open a vault in a browser using the web client.
- Useful for quick access without installing the desktop app.

## Offline First
- Local vaults keep working when the network is down.
- Remote vaults depend on the server for sync and access.
