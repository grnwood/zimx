- Your notes are plain files on your machine.
- Everything works offline by default.
- AI features are opt-in and explicit.
- Links, tags, tasks, and search work together.

## Link Format
ZimX uses wiki-style links like this:

`[:Page_Name|Nice label]`

If a page has spaces, use underscores in the link target. ZimX resolves it back to spaces.
