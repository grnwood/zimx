Add the following:
ctrl-t "verbatim"  buts selection in backticks for fixed length render.
ctrl-(1-5) called "Heading {x}" will convert current line to heading tags (#) this will edit the current line to have appropriate heading level (#) applied.
ctrl-7 should be "Remove Heading" this will remove heading markers from current line.

Change highlight to use ctrl-u

Add:
"Bullet List"  turns the selected lines into bullet list, keeps indenting levels.
"Dash List" turns selected lines into dash list, keeps indenting,
"Checkbox List" turns selected lines into checkboxes, keeps indenting.
"Clear List" clears the list of any type
        	
 ctrl-9 is "Clear Formatting"  restores selection to plain text.
